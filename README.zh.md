## 顶尖大学

### 第一梯队 - 前帝国大学

- **[东京大学 (東京大学)](https://www.u-tokyo.ac.jp/en/)** - 日本最负盛名的大学,日本排名始终第一
- **[京都大学 (京都大学)](https://www.kyoto-u.ac.jp/en)** - 以培养诺贝尔奖获得者和前沿研究而闻名
- **[大阪大学 (大阪大学)](https://www.osaka-u.ac.jp/en)** - 日本西部领先的研究型大学
- **[东北大学 (東北大学)](https://www.tohoku.ac.jp/en/)** - 材料科学和工程领域的先驱
- **[名古屋大学 (名古屋大学)](https://en.nagoya-u.ac.jp/)** - 汽车工程和物理学领域实力强劲
- **[九州大学 (九州大学)](https://www.kyushu-u.ac.jp/en/)** - 日本南部的领先大学
- **[北海道大学 (北海道大学)](https://www.global.hokudai.ac.jp/)** - 日本最大的校园,农业和环境科学实力强劲

### 第二梯队 - 顶尖国立大学

- **[东京工业大学 (東京工業大学)](https://www.titech.ac.jp/english)** - 顶尖工程技术大学
- **[一桥大学 (一橋大学)](https://www.hit-u.ac.jp/eng/)** - 经济和商业领域的顶尖机构
- **[筑波大学 (筑波大学)](https://www.tsukuba.ac.jp/en/)** - 领先的科学研究型大学
- **[神户大学 (神戸大学)](https://www.kobe-u.ac.jp/en/)** - 商业和国际关系实力强劲
- **[东京医科齿科大学 (東京医科歯科大学)](https://www.tmd.ac.jp/english/)** - 顶尖医学院

### 顶尖私立大学

- **[庆应义塾大学 (慶應義塾大学)](https://www.keio.ac.jp/en/)** - 日本最古老的私立大学,商业课程享有盛誉
- **[早稻田大学 (早稲田大学)](https://www.waseda.jp/top/en/)** - 高度国际化,人文和社会科学实力强劲

## 申请资源

### 考试系统

#### EJU (日本留学试验)
- 大多数本科入学必需
- 每年提供两次(6月和11月)
- 提供日语和英语版本
- [EJU官方网站](https://www.jasso.go.jp/en/eju/index.html)

#### 各大学单独考试
- 每所大学都进行自己的入学考试
- 可能包括笔试、面试和作品集审查
- 要求因课程和大学而异

### 申请门户
- [JASSO 日本留学门户](https://www.studyinjapan.go.jp/en/)
- [日本留学支援](https://www.jpss.jp/en/)
- [JREC-IN门户](https://jrecin.jst.go.jp/seek/SeekTop?ln=1) - 研究职位

### 要求清单
- [ ] 有效护照
- [ ] 学术成绩单
- [ ] 毕业证书
- [ ] 语言能力证明(JLPT或TOEFL/IELTS)
- [ ] EJU成绩(如需要)
- [ ] 目的陈述
- [ ] 推荐信
- [ ] 财务证明
- [ ] 在留资格认定证明书(COE)

## 学生生活

### 住宿
- **大学宿舍** - 最实惠,名额有限
- **合租房** - 国际学生中很受欢迎
- **私人公寓** - 更贵但更独立
- **寄宿家庭** - 文化沉浸机会

### 兼职工作
- 国际学生每周可工作最多**28小时**(假期期间40小时)
- 常见工作:英语教学、便利店、餐厅
- 需要许可从事学习以外的活动

### 学生组织
- 社团活动(サークル) - 体育、文化、爱好
- 国际学生协会
- 学术俱乐部和研究小组

### 健康保险
- 所有居民都必须购买国民健康保险(NHI)
- 覆盖70%的医疗费用
- 学生每月保费约¥2,000-3,000

- N5(初级)到N1(高级)
- 本科课程通常要求N2
- 日语研究生课程通常要求N1
- 每年举行两次(7月和12月)
