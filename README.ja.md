# Awesome 日本の大学 [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> 日本の高等教育機関、リソース、情報の厳選リスト

日本には世界でも有数の名門大学があり、工学・技術から芸術・人文科学まで、幅広い分野で優れた教育を提供しています。このガイドは、日本の大学への進学を検討している留学生や国内学生のための包括的な情報を提供します。

## トップ大学

### 第一階層 - 旧帝国大学

- **[東京大学](https://www.u-tokyo.ac.jp/)** - 日本で最も権威のある大学、日本国内で常に第1位
- **[京都大学](https://www.kyoto-u.ac.jp/)** - ノーベル賞受賞者を多数輩出し、最先端の研究で知られる
- **[大阪大学](https://www.osaka-u.ac.jp/)** - 西日本を代表する研究大学
- **[東北大学](https://www.tohoku.ac.jp/)** - 材料科学と工学の先駆者
- **[名古屋大学](https://www.nagoya-u.ac.jp/)** - 自動車工学と物理学に強い
- **[九州大学](https://www.kyushu-u.ac.jp/)** - 南日本の主要大学
- **[北海道大学](https://www.hokudai.ac.jp/)** - 日本最大のキャンパス、農学と環境科学に強い

### 第二階層 - トップ国立大学

- **[東京工業大学](https://www.titech.ac.jp/)** - 工学・技術系のトップ大学

- **[筑波大学](https://www.tsukuba.ac.jp/)** - 科学・研究の主要大学
- **[神戸大学](https://www.kobe-u.ac.jp/)** - ビジネスと国際関係に強い
- **[東京医科歯科大学](https://www.tmd.ac.jp/)** - トップ医学部

### トップ私立大学

- **[慶應義塾大学](https://www.keio.ac.jp/)** - 日本最古の私立大学、名門ビジネスプログラム

## 出願リソース

### 試験制度

#### EJU(日本留学試験)
- ほとんどの学部入学に必要
- 年2回(6月と11月)実施
- 日本語と英語で提供
- [EJU公式サイト](https://www.jasso.go.jp/ryugaku/eju/index.html)

#### 各大学の個別試験
- 各大学が独自の入学試験を実施
- 筆記試験、面接、ポートフォリオ審査を含む場合がある
- プログラムと大学により要件が異なる

### 出願ポータル
- [JASSO 日本留学ゲートウェイ](https://www.studyinjapan.go.jp/)
- [日本留学支援](https://www.jpss.jp/)
- [JREC-INポータル](https://jrecin.jst.go.jp/) - 研究職向け

### 必要書類チェックリスト
- [ ] 有効なパスポート
- [ ] 成績証明書
- [ ] 卒業証明書
- [ ] 語学能力証明(JLPTまたはTOEFL/IELTS)
- [ ] EJUスコア(必要な場合)
- [ ] 志望理由書
- [ ] 推薦状
- [ ] 経済証明
- [ ] 在留資格認定証明書(COE)

## 学生生活

### 宿泊施設
- **大学寮** - 最も手頃、空室は限られている
- **シェアハウス** - 留学生に人気
- **民間アパート** - より高価だが独立性がある
- **ホームステイ** - 文化交流の機会

### アルバイト
- 留学生は**週28時間まで**働くことができる(休暇中は40時間)
- 一般的な仕事:英語教師、コンビニ、レストラン
- 資格外活動許可が必要

### 学生組織
- サークル活動 - スポーツ、文化、趣味
- 留学生会
- 学術クラブと研究グループ

### 健康保険
- すべての居住者に国民健康保険(NHI)が必要
- 医療費の70%をカバー
- 学生の月額保険料は約2,000〜3,000円

## 語学学校

大学入学前に日本語能力を向上させたい方のために:

### トップ語学学校
- **[KCP国際日本語学校](https://www.kcpinternational.com/)** - 東京
- **[ISI日本語学校](https://www.isi-global.com/)** - 複数の場所
- **[吉田日本語学院](https://www.jpschool.ac.jp/)** - 東京
- **[京都日本語学校](https://www.kyotojls.co.jp/)** - 京都
- **[福岡外語専門学校](https://www.fflc.ac.jp/)** - 福岡

### JLPT(日本語能力試験)
- N5(初級)からN1(上級)まで
- 学部プログラムには通常N2が必要
- 日本語での大学院プログラムにはN1が必要な場合が多い
- 年2回(7月と12月)実施
